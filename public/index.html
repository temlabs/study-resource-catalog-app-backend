My server API

RESTful JSON API

GET Requests

  • Retrieve resources sorted by post date in descending order

    • GET /resources

  • Gets all users

    • GET /users

  • Gets comments for a given resource

    • GET /comments/:resource_id

  • Gets all reactions for a given resource

    • GET /reactions

  • Gets all content-types

    • GET /content-types

  • *Gets all study tips for a specific users

    • GET /study-list/:user_id

POST Requests

  • Adds resource to a database

    • POST /resources

  • *Post comments for a particular resource

    • POST /comments

  • *Post a reaction for a given resource

    • POST /reactions/:resource_id

  • *Post a reactions for a given resource

    • POST /reactions/:resource_id

  • *Post a resource to study list for a specific users

    • GET /study-list
